⭐ Primary Featured Project · Undergraduate Thesis

Intelligent Academic Resource Management System

An evolved, AI-powered academic resource platform built for Palawan State University's College of IT, developed as a significant improvement over Buzy Reviewer. Students and faculty can upload documents and receive AI-generated summaries, flashcards, quizzes, and key concept breakdowns. The full backend, powered by Ollama (Llama 3 / Mistral via FastAPI), runs only within the local development environment. The live GitHub deployment provides the frontend interface only, with GPT-4o Mini API planned for the full production build.
